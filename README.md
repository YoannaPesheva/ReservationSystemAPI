# Система за управление на резервации на зали

Проект за курса "Програмиране с Пайтън" - 2025/2026 година, ФМИ

# Как да стартираме проекта локално?
1.	Изпълняват се следните команди в терминала:
   - git clone https://github.com/YoannaPesheva/ReservationSystemAPI
   - pip install –r requirements.txt
   - uvicorn main:app –reload
3.	Следният линк се поставя в браузъра: http://127.0.0.1:8000/docs

# Описание на проекта
Проектът представлява платформа за управление на резервации на зали, в която има три типа потребители – нерегистрирани, обикновени потребители, наемодатели и админ. Основните функционалности, от които потребителите могат да се възползват, са: регистрация и управление на профила си, търсене и филтриране по зададени критерии, създаване на резервации, оставяне на оценка и коментар, както и добавяне към любими места. Наемодателите пък имат възможността да променят статуса на резервацията и да управляват своите зали (включително и да ги изтриват). Заявките за резервация успешно преминават към статут на изчакване само ако времето, зададено при създаването на заявката, не съвпада с друга вече направена регистрация. Платформата разполага и с админ, който може да управлява цялата система.

Използваните технологии са FastAPI, SQLAlchemy.
